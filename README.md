# Corrona App

A Python-based application to track and display COVID-19 statistics, including cases, recoveries, and vaccination data, with real-time updates and a user-friendly interface.

---

## Features

- View real-time COVID-19 statistics by country or globally.
- Track confirmed cases, recoveries, and deaths.
- Display vaccination data.
- Simple and interactive command-line interface (or GUI if implemented).
- Easy to update and extend with new data sources.
